# Implementation Plan: Place Data Service

**Branch**: `001-place-data-service` | **Date**: 2026-04-16 | **Spec**: [spec.md](spec.md)

---

## Summary

Build a standalone FastAPI-based data service for the Chito-Go project that ingests Google Places JSON payloads, normalizes them into an internal source-agnostic schema, stores both normalized records and raw payloads in PostgreSQL, and exposes simple REST endpoints for retrieval. The service is a data layer only — no auth, chat, itinerary, or frontend logic.

engine = create_engine(settings.DATABASE_URL, pool_pre_ping=True)

    db = SessionLocal()

    finally:

### app/services/ingestion.py

Key logic:

```python
import unicodedata
from datetime import datetime, timezone
from sqlalchemy.orm import Session
from app.models.place import Place
from app.models.place_source_google import PlaceSourceGoogle
from app.models.place_features import PlaceFeatures

def _safe_get(d: dict, *keys, default=None):
    """Safely traverse nested dict keys."""
    for key in keys:
        if not isinstance(d, dict):
            return default
        d = d.get(key, default)
    return d

def _normalize_name(name: str) -> str:
    return unicodedata.normalize("NFKC", name).lower().strip()

def _extract_district(payload: dict) -> str | None:
    for component in _safe_get(payload, "addressComponents", default=[]):
        if "sublocality" in component.get("types", []) or "locality" in component.get("types", []):
            return component.get("longText")
    return None

def ingest_google_place(db: Session, payload: dict, features: dict | None = None) -> dict:
    google_place_id = payload.get("id")
    if not google_place_id:
        raise ValueError("google_place_id is required")

    # Always store raw payload (unconditional once google_place_id is present)
    raw_record = PlaceSourceGoogle(
        google_place_id=google_place_id,
        raw_json=payload,
        fetched_at=datetime.now(timezone.utc),
    )

    display_name_raw = _safe_get(payload, "displayName", "text") or payload.get("displayName")
    if not display_name_raw or not isinstance(display_name_raw, str):
        # Store raw only; skip normalized record
        db.add(raw_record)
        db.commit()
        db.refresh(raw_record)
        return {
            "google_place_id": google_place_id,
            "action": "raw_only",
            "detail": "display_name is required for normalized record; raw payload stored only",
        }

    place_data = {
        "google_place_id": google_place_id,
        "display_name": display_name_raw,
        "normalized_name": _normalize_name(display_name_raw),
        "primary_type": payload.get("primaryType"),
        "types_json": payload.get("types"),
        "formatted_address": payload.get("formattedAddress"),
        "district": _extract_district(payload),
        "latitude": _safe_get(payload, "location", "latitude"),
        "longitude": _safe_get(payload, "location", "longitude"),
        "rating": payload.get("rating"),
        "user_rating_count": payload.get("userRatingCount"),
        "price_level": payload.get("priceLevel"),
        "business_status": payload.get("businessStatus"),
        "google_maps_uri": payload.get("googleMapsUri"),
        "website_uri": payload.get("websiteUri"),
        "national_phone_number": payload.get("nationalPhoneNumber"),
        "opening_hours_json": payload.get("regularOpeningHours"),
        "last_synced_at": datetime.now(timezone.utc),
    }

    existing = db.query(Place).filter_by(google_place_id=google_place_id).first()
    if existing:
        for k, v in place_data.items():
            setattr(existing, k, v)
        place = existing
        action = "updated"
    else:
        place = Place(**place_data)
        db.add(place)
        action = "created"

    db.flush()  # get place.id before raw record insert

    raw_record.place_id = place.id
    db.add(raw_record)

    # Optional features upsert
    if features:
        existing_features = db.query(PlaceFeatures).filter_by(place_id=place.id).first()
        if existing_features:
            for k, v in features.items():
                if hasattr(existing_features, k):
                    setattr(existing_features, k, v)
        else:
            db.add(PlaceFeatures(place_id=place.id, **{
                k: v for k, v in features.items()
                if k in PlaceFeatures.__table__.columns.keys()
            }))

    db.commit()
    db.refresh(place)
    return {"place_id": place.id, "google_place_id": google_place_id, "action": action}
```

### app/routers/places.py

Three endpoints in one file:

**POST /api/v1/places/import/google**:
- Request body: `{ "payload": dict, "features": dict | None }`
- Call `ingestion.ingest_google_place(db, payload, features)`
- Return result dict; raise `HTTPException(422)` on `ValueError`

**GET /api/v1/places**:
- Query params: `district`, `primary_type`, `indoor` (bool), `budget_level`, `min_rating` (float), `limit` (int, default 50), `offset` (int, default 0)
- Build SQLAlchemy query on `Place`, apply filters with `if param is not None`
- Return list of dicts (use Pydantic schema or `row.__dict__`)

**GET /api/v1/places/{place_id}**:
- Query Place by id; 404 if not found
- Eagerly load or separate-query PlaceFeatures by place_id
- Return full detail dict including `features` block or `null`

Safe to run multiple times (idempotent).

    finally:
        db.close()
    print("[seed] Done.")

if __name__ == "__main__":
    main()

## Assumptions

1. The PostgreSQL database and user already exist and are accessible at the configured URL.
2. `Base.metadata.create_all()` is the only schema management needed for this milestone; Alembic is deferred.
3. The Google Places (New) API shape is the initial source format. The ingestion service maps specific field paths from that shape, but the internal schema is source-agnostic (FR-011).
4. `district` is derived from `addressComponents` at ingest time; if not present in the payload, it defaults to `None`.
5. `indoor`, `outdoor`, and `budget_level` are not derivable from Google Places payloads reliably — they default to `None` at ingest and will be enriched by a separate process later.
6. `trend_score` and `confidence_score` are not computed by this service; they default to `None` and will be written externally.
7. The seed script imports from `app/` using a `sys.path` adjustment; this avoids needing a package install step.
8. No authentication middleware is added; all endpoints are open on the configured host/port.
9. `pool_pre_ping=True` on the SQLAlchemy engine handles transient connection drops gracefully.

## Risks and Gotchas

| Risk | Impact | Mitigation |
|---|---|---|
| `displayName` field may be a dict `{"text": "...", "languageCode": "..."}` or a plain string in some Google payload variants | Ingestion fails or produces wrong name | `_safe_get(payload, "displayName", "text") or payload.get("displayName")` handles both shapes |
| All models must be imported before `create_all` is called | Tables not created on startup | `import app.models` in `main.py` before `create_all` (not just individual model files) |
| SQLAlchemy 2.x uses `DeclarativeBase` (not `declarative_base()`) | Import error if mixing 1.x and 2.x styles | Pin `sqlalchemy>=2.0.0` and use `class Base(DeclarativeBase): pass` |
| `pydantic-settings` v2 uses `model_config` not inner `Config` class | `SettingsConfigDict` import required | Use `model_config = {"env_file": ".env"}` directly on the class |
| `place_features` uses `place_id` as PK — SQLAlchemy needs `primary_key=True` on the FK column | ORM error if not set correctly | Explicitly set `primary_key=True` on the `place_id` column in the model |
| `onupdate=func.now()` on `updated_at` only fires on ORM-level updates, not raw SQL | `updated_at` not updated when using `db.execute()` directly | Always use ORM objects (`setattr`) for updates; avoid raw SQL updates on Place rows |
| `Numeric` type requires `asdecimal=False` if you want float in JSON responses | Pydantic serialization may fail on `Decimal` | Set `asdecimal=False` on Numeric columns or use `model_config = {"arbitrary_types_allowed": True}` in Pydantic schemas |
| Scripts run from repo root — `app/` must be on `sys.path` | `ModuleNotFoundError` in seed script | `sys.path.insert(0, repo_root)` at top of script |
| Re-running `POST /import/google` with identical payload appends a new `place_source_google` row each time | Table grows on repeated testing | Expected behavior (append-only retention); document clearly |
